This repository contains a comprehensive overview and a code space for object detection using RCNN, FastRCNN, Faster-RCNN and mask-RCNN(not implemeted yet). <br>


For RCNN object detection I used the data set exsisting in the kaggele about car and trucks. For the faster RCNN I used  Aquarium Dataset from Roboflow [click Here](https://public.roboflow.com/object-detection/aquarium/2)

## Installation

1. Clone the project.
```bash
git clone "https://github.com/nilum2002/Object-Detection-Using-RCNN" 
```
2. Install required dependencies:
```bash
pip install -r requirements.txt
```
3. Create env and test the code
